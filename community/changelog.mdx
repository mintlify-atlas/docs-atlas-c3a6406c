## [2.0.0-beta9]

### Fixed
- Stats on official servers not updating
- "The title is installed in a language you are not entitled to play" for some users
  - Module path moved to `C:\ProgramData\Kyber\Module` to prevent issues with non-ASCII characters
- Issue where it was not possible to download mods for a full server
- All instances of a server group showing the same map in the server browser
- Launcher greying out when a server group was full
- JSON file map rotation imports
- "CREATE NEW COLLECTION" button greying out on the server host page
- Stats page softlocking when failing to load EA stats
- All content disappearing when Maxima failed to start the game
  - Added warning dialog when trying to start a game with an expired EA session

### Added
- New filter options to the mods page
  - Filter mods by gameplay and cosmetic categories
- Option to set custom icons for collections
- Copy server link buttons

### Changed
- **Forced HDR to be always disabled**

<Warning>
  HDR is now permanently disabled due to compatibility issues with the Kyber module.
</Warning>

---

## [2.0.0-beta8]

### Major Changes
- Changed how clients authenticate with servers
  - Fixes an issue where it was possible to join full servers
- New CDN for mod downloads
  - Should improve download speeds for most users
  - Reduces issues with downloads failing

### Fixed
- Crash regarding multiple levels being loaded when using Kyber.Restart
- **Forced HDR to be always disabled**
- 1 player bug (for the 4th time)

### Added
- Extracting progress now shown when installing mods
  - Only works for .zip and .rar files
  - .7z files won't show progress
- Major overhaul of the download manager

<Note>
  If you encounter any issues with downloads after this update, please report them in our [Discord server](https://discord.gg/kyber).
</Note>

---

## [2.0.0-beta7]

### Major Features
- **Discord linking**
  - Link your Discord account to your Kyber account in settings
  - Patreon supporters have Discord linked automatically after login
- **Proximity voice chat improvements**
  - Push to talk option added
  - Switch input/output devices without being in-game
  - In-game icons for proximity voice chat

### Fixed
- Launcher failing to check if user is logged in to Nexus Mods
- Proximity voice chat not working
- Game sometimes crashing when using `Kyber.Restart`
- Launcher getting stuck at "Updating Module"
- Stats on official servers not updating
- Mods list on host page not refreshing after filesystem change

### Added
- Updated join modal UI
- Collections can now be saved by pressing `Enter` in the name input
- Files in the `MISCELLANEOUS FILES` category now displayed in mod browser

### Changed
- Proxies now pinged multiple times for more accurate ping values
- EA usernames should update again on KYBER after name change
  - Can take up to 7 days for username to update
- Launcher now clears Nexus Mods cache between versions
- Mod browser cache moved to system temp folder

---

## [2.0.0-beta6]

### Major Features
- **Report system**
  - Report users directly from in-game panel (press `INSERT`)
  - Report users via profile search
- **Custom modes and maps support**
  - Map rotation builder now shows custom maps/modes
  - Mods must implement the new format
- **New server grouping system**
  - Official servers grouped together
  - Most popular instance selected by default
  - Manual instance selection available in join dialog

### Fixed
- Downloads failing with "Request Range Not Satisfiable"
- Setting paths with non-ASCII characters as mod path
- Creating new Nexus Mods account in Launcher
- Launcher turning invisible on startup
- After name change, users unable to promote other players to moderators
- EA usernames not updating on KYBER (can take up to 3 days)
- (Maxima) Cloud Sync - fixes Battlefront 2 settings resetting after every game start

### Added
- Warning when trying to use more than 247 mods

<Warning>
  Using more than 247 mods may cause stability issues and crashes.
</Warning>

## [2.0.0-beta5]

### Fixed
- **Proxy reconnection bug** - packets sent while reconnecting now reported as successful
- Launcher not able to search for mods in Mod Browser
- Launcher sometimes crashing while starting the game
  - Windows 7 Compatibility Mode no longer needed
- Promoting a player to moderator not working
- Launcher not able to load proxies
- Launcher loading wrong files when using a mod collection
- Launcher not starting correctly and displaying white screen
- Launcher sometimes failing to download mods with Nexus Mods Premium
- Launcher turning grey when trying to join a server
- Memory leak in the Module

### Added
- Region icons to server browser
- Server password inputs now sensitive
- Option to disable proximity chat
- Option to shuffle teams between rounds (host setting)
- Option to unlink or link Patreon account in settings
- Warning when multiple big mod packs are used together
- Usernames now displayed for punishments and moderators
- Frosty Collection screenshots shown when inspecting them
- (Maxima) Improved error messages

### Changed
- Potential fix for Patreon account linking issues
- Removed "PLAY" button from server browser (EA servers still broken)
  - To play offline, go to Mods page, select collection, and click "PLAY"

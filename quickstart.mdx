## Get up and running quickly

This quickstart guide will help you download Kyber, install it, and join your first community server.

<Steps>
  <Step title="Download the Launcher">
    Download the latest Kyber Launcher from [kyber.gg](https://kyber.gg).
    
    <Note>
      Make sure you have Star Wars Battlefront II (2017) installed before proceeding.
    </Note>
  </Step>

  <Step title="Install and Launch">
    Run the installer and follow the setup wizard. The installer will:
    - Install the launcher to `C:\Program Files\KYBER Launcher`
    - Set up the game module in `C:\ProgramData\Kyber`
    - Create file associations for `.kbcollection` files
    
    Launch Kyber after installation completes.
  </Step>

  <Step title="First-Time Setup">
    On first launch, Kyber will guide you through initial setup:
    
    1. **EA Account**: Sign in with your EA account (required to play)
    2. **Game Detection**: Kyber will automatically locate your Battlefront II installation
    3. **Mod Directory**: Choose where to store downloaded mods
    4. **Optional Integrations**:
       - Link your Nexus Mods account (for mod downloads)
       - Link your Discord account (for community features)
       - Link your Patreon (if you're a supporter)
  </Step>

  <Step title="Join a Server">
    You're ready to play!
    
    1. Click **Server Browser** in the main navigation
    2. Browse available servers or use filters to find your preferred game mode
    3. Click on a server to see details, including:
       - Current players and map
       - Required mods (if any)
       - Server rules
    4. Click **Join Server**
    5. If the server requires mods you don't have, Kyber will offer to download them automatically
    6. Click **Play** to launch the game and connect
    
    <Info>
      Kyber will handle mod installation, game launching, and server connection automatically.
    </Info>
  </Step>
</Steps>

## Troubleshooting

Having issues? Check these common problems:

<AccordionGroup>
  <Accordion title="Launcher won't start or shows white screen">
    This was a known issue in beta versions. Make sure you're running the latest version from kyber.gg.
    
    If the problem persists:
    1. Close Kyber completely
    2. Delete `C:\Users\<YourUsername>\AppData\Roaming\kyber_launcher`
    3. Restart the launcher
  </Accordion>

  <Accordion title="Can't connect to servers">
    Check that:
    - You're signed in with your EA account
    - Star Wars Battlefront II is installed and updated
    - Your firewall isn't blocking Kyber
    - The server you're trying to join isn't full
  </Accordion>

  <Accordion title="Mods not loading">
    1. Verify your mod directory path doesn't contain special or non-ASCII characters
    2. Check that mods are compatible with your game version
    3. Try disabling all mods and re-enabling them one at a time
    4. Note: There's a limit of 247 mods - the launcher will warn if you exceed this
  </Accordion>
</AccordionGroup>

For more detailed troubleshooting, see the [Troubleshooting Guide](/community/troubleshooting).
